Url Shortener Microservice

Enter the URL you would like to shorten at the /new/ endpoint

EXAMPLE: ~/new/http://www.google.com

Will return a json object like this: 

            
{
    "fullLink": "http://www.google.com",
    "shortLink": "~/sh/123"
}

Read more about this project here. https://www.freecodecamp.com/challenges/url-shortener-microservice
